# Lap 1 Code Challenge

## Installation & Usage
### Installation
- Clone or download the repo
- Open terminal and navigate to the cloned folder
- Run `npm install` to install dependencies

### Usage
- Run `npm start` to launch server or `npm run dev` if using nodemon
